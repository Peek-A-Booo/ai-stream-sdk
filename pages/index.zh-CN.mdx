# AI Stream SDK

**AI Stream SDK** 是一个插件，能够帮助你快速实现 AI API 数据流。它支持各种模型的 API 转发，使用它可以让你迅速搭建自己的 API 转发服务。

## 功能特性

- 数据流转发
- 提供流数据响应各个阶段的监控
- [统一数据流响应结束标识](/tips#统一数据流响应结束标识)
- _未来支持对模型响应的 token 额度计算_

## 模型支持

逐渐增加对各个模型的支持。
